Rezonator

Desktop app aimed at analyzing spoken data for coreference, dialogic resonance, and dependency structure. Some automatic annotation may be done with the R package RezonateR.

Release year: 2021

Paper: http://semdial.org/anthology/Z20-Bois_semdial_0031.pdf

Code: https://github.com/johnwdubois/rezonator

Availability: open-source

Data types: tags, time-aligned, chunk-spans, overlapping-spans, dependency-tree, graph

Import formats: plaintext, json, conllu, csv

Export formats: json, csv

Keywords: multilayer, discourse, coreference, resonance
